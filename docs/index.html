System Overview

Desktop application for viewing, processing, and analyzing RAW and standard-format images. Uses OpenCV DNN (YuNet model) for face detection, u2net ONNX for background removal, and GLSL shaders for image adjustment operations through a Qt6 UI with OpenGL rendering.

Key Features

  • GPU-accelerated real-time image adjustments via OpenGL shaders (exposure, contrast, gamma, temperature, grayscale, detail enhancement)
  • Multi-strategy face detection using OpenCV DNN YuNet with adaptive thresholds, CLAHE enhancement, extended rotation, weighted box fusion, NMS
  • Background removal using u2netp ONNX model via OpenCV DNN
  • Non-destructive editing with full undo/redo stack (EditStack, EditOperation)
  • RAW camera format support via LibRaw with auto-rotation, metadata extraction
  • LUT color grading (CubeLUT .cube format with trilinear interpolation)
  • Split-screen before/after comparison with draggable divider
  • GPU compute shader histogram calculation
  • Plugin architecture for extensible image processor and face detector backends
  • Asynchronous multi-threaded image loading with LRU caching

Core Modules

ModuleTypeResponsibility
OrchestratorCoreCentral facade coordinating all subsystems. Manages plugin lifecycle, provides unified sync/async APIs for image processing, face detection, and background removal. Falls back to default implementations when plugins are unavailable.
ImageProcessorCoreCPU-based image processing: exposure, contrast, brightness, gamma, temperature, tint, saturation, vibrance, highlights/shadows, detail enhancement, grayscale conversion, LUT color grading.
FaceDetectorCoreMulti-strategy face detection using OpenCV DNN (YuNet ONNX model). Supports adaptive thresholds, CLAHE enhancement, forced upscale, extended rotation. Applies NMS and post-processing (bounding box expansion, aspect ratio correction, size/position filtering).
SegmenterCoreAI image segmentation using OpenCV DNN with u2netp ONNX model. Returns segmentation mask and composited image.
ProcessingPipelineCoreAsync pipeline state machine (Loading, FaceDetection, ImageProcessing, Completed, Error) using QtConcurrent. Includes coroutine-based AsyncImageProcessor.
EditStackCoreUndo/redo stack managing EditOperation instances. Supports push, undo, redo, clear, batch apply-all.
LUTCoreCubeLUT struct parsing .cube format files. Trilinear interpolation for 3D color lookup tables.
PluginManagerPluginPlugin discovery, loading (QPluginLoader), lifecycle, dependency resolution, performance monitoring.

Face Detection Pipeline

  • Image characteristics analysis for adaptive strategy selection
  • Adaptive confidence thresholds based on image quality
  • Primary detection: DNN-based with YuNet ONNX model
  • Fallback strategies: enhanced CLAHE, forced upscale, extended rotation, original size
  • Detection score calculation: size, position, aspect ratio heuristics
  • Non-Maximum Suppression (IOU threshold: 0.3, keep highest confidence)
  • Post-processing: bounding box expansion (10px), aspect ratio correction (1.3-1.5 target), size filtering (5%-80% of image), position filtering (upper portion bias)
